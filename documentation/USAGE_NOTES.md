# Usage Notes

The original workbooks are Excel 97-era `.xls` files. They are preserved in `code/original/` and `data/original/`. LibreOffice `.xlsx` conversions are provided in `code/modernized/` and `data/modernized/` so the workbook sheets and formulas can be inspected with current tools.

The main paper-matching model workbook is `code/original/market2.1.xls`. The base cost-benefit model and example workbooks are `code/original/AUTOMAT.XLS` and `code/original/Example.xls`. The Carquinez Bridge traffic-count input is `data/original/CarquinToll.xls`.

Embedded VBA/XLM text was extracted where present. The `.xlsx` conversions are inspection copies and may not preserve macro behavior; use the original `.xls` files as the provenance record.
